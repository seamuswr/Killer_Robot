Detailed Description

The original file has been modified to run a cooperative multitasking program.

Task 1 rotates a the robot 180Ëš and handles fine adjustments, task 2 pulls the trigger on the turret, and task 3 operates the camera. Data is shared cooperatively. The camera task waits 5 seconds, takes a picture, and sends the rotation task the amount to adjust. Once adjusted, the rotation task tells the trigger task to fire.
